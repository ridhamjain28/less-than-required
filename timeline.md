# Timeline

## v1.1.0 - The "Anywhere" Update (Current)
* **Manual Processing UI**: Added a comprehensive popup interface allowing users to manually process files.
* **Smart Presets**: One-click presets for common government form requirements:
  * **Passport Photo**: 200x230, Max 50KB, JPG
  * **Signature**: 140x60, Max 20KB, JPG
  * **Document**: Max 2MB, PDF
* **Custom Settings**: Fine-grained control over target size (KB), dimensions, and output format.
* **Privacy First**: 100% offline processing. No files are ever sent to an external server.
* **Universal Compatibility**: Allows users to download the compressed file and manually upload it, bypassing restrictive government websites that block automatic interceptions.

## v1.0.0 - Initial Release
* **Auto-Compression**: Invisible, automatic file compression when dragging and dropping files onto supported websites.
* **Drag-and-Drop Interception**: Intercepts file drops, compresses them on the fly, and seamlessly passes them to the website.
* **Basic Formats Supported**: Initial support for common image formats (JPEG, PNG).
